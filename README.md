# Game Room

A single-page menu that links to my serverless, peer-to-peer browser games.
No accounts, no backend — every game runs in the browser and connects players
directly over MQTT.

**Live:** https://lovro-git.github.io/games/

## Games

- [Texas Hold'em](https://lovro-git.github.io/poker/) — No-Limit Hold'em with side pots, cash & tournament modes.
- [Belot](https://lovro-git.github.io/belot/) — Croatian trick-taking card game with full rules.

## Adding a game

Edit `index.html` and copy one of the `<a class="game">` blocks in `<main>`,
pointing `href` at the new game's GitHub Pages URL.
